# PLATOON
3D HTML FPS endless survival game

PLATOON is a 3D FPS game created in HTML, mostly by Claude AI. The game is endless, where you fight waves of enemies that shoot back. Features include:
- ADS
- Canted aiming
- Working recoil

# CONTROLS
WASD - movement
Q/E - cant left/right
[SPACE] - jump
Left/Right Click - Shoot/ADS
R - reload
